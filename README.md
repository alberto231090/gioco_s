# Babysitter Simulator (prototipo)

Progetto: simulatore 3D realistico per PC in cui interpreti un babysitter. Questo repository contiene la scaffolding iniziale del progetto e una lista di asset/attività da svolgere.

## Obiettivi principali
- Prototipo PC (consigliato: Unity HDRP)
- Interazioni realistiche non violente (consolare, cambiare pannolino, time-out, routine)
- Import asset FBX gratuiti per personaggi (bambini, genitori)
- Modalità cooperativa locale

## Struttura iniziale
- `Assets/Models` - modelli FBX importati
- `Assets/Scripts` - script C# (placeholder)
- `docs` - documentazione e lista asset

## Prossimi passi
1. Ricerca e download di 10 modelli FBX gratuiti e verifica licenze
2. Import nel progetto Unity e scena di prova
3. Implementazione meccaniche base e UI

**Nota sulla compliance:** il gioco esclude qualsiasi contenuto sessuale o dannoso verso minori; tutte le meccaniche di disciplina sono educative e non fisiche.

Mantieni questo README aggiornato con note su asset e licenze.
